# Product: CaddyAI

## Overview
CaddyAI is a desktop AI assistant designed to streamline developer workflows by integrating with essential tools like Linear and Slack.

## Core Features
- **Smart Integrations:** Seamlessly connects with Linear for issue tracking and Slack for communication.
- **Secure Architecture:** Prioritizes security with robust credential management using macOS Keychain.
- **User-Centric Design:** Features an intuitive onboarding flow and a dedicated settings interface for managing integrations.
- **AI-Powered:** Leverages OpenAI's language models to provide intelligent assistance and automation.
