# Tech Stack

## Frontend (macOS)
- **Language:** Swift 5.9+
- **Framework:** SwiftUI
- **Target:** macOS
- **Key Libraries:** Security.framework (Keychain)

## Backend
- **Language:** Python
- **Environment:** Virtual Environment (.venv)
- **Dependencies:** OpenAI API, Linear API, Slack API

## Data & State
- **Secure Storage:** macOS Keychain (for API keys and secrets)
- **Local Storage:** UserDefaults (for non-sensitive preferences)
