# caddyAI Setup

Use this guide to go from fresh install to a working caddyAI with API access.

## Steps
1. Open Settings from the menu bar tray icon.
2. General: choose appearance and enable Microphone + Accessibility permissions.
3. API & Accounts: add your API key (and Base URL if self-hosted), Save, then Test connection.
4. Integrations (optional):
   - Slack: add a bot token with required scopes, Connect, then Test.
   - Linear: add a personal API key and team key, Connect, then Test.
   Tokens are stored locally in Keychain.
5. Setup: run the onboarding wizard or skip and return later.

## Checklist
- Settings opens
- Permissions enabled
- API key saved and tested
- Optional integrations connected
- Onboarding complete
